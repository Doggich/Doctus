# Doctus — Markdown редакто

**Doctus** — это мощный Markdown-редактор, разработанный на WPF и .NET 8. Позволяет создавать, редактировать и просматривать Markdown-документы с мгновенным предпросмотром.

## Возможности

### Редактирование:
- Поддержка стандартного синтаксиса `Markdown`
- Вставка тегов форматирования через меню или горячие клавиши
- Живой предпросмотр в реальном времени
- Подсветка синтаксиса кода с помощью `highlight.js`
- Кнопки копирования для блоков кода

### Работа с файлами:
- Создание новых документов (`Ctrl+N`)
- Открытие `Markdown` файлов (`Ctrl+O`)
- Открытие `HTML` файлов
- Сохранение в форматах: `.md`, `.html`, `.pdf`
- Экспорт в Markdown, `HTML` и `PDF`
- Отслеживание несохранённых изменений

### Отправка по email
- Встроенный почтовый клиент с поддержкой `SMTP`
- Отправка документов как вложение в форматах `Markdown`, `HTML` или `PDF`
- Красивое `HTML-оформление` писем
- Поддержка различных почтовых сервисов

### Настройка внешнего вида
- Три режима просмотра: редактор, предпросмотр, оба вместе
- Настраиваемые CSS-стили для предпросмотра
- Загрузка пользовательских стилей из файла
- Сброс стилей к стандартным
- Тёмная тема по умолчанию

### Горячие клавиши

| Сочетание | Действие |
|-----------|----------|
| `Ctrl+N` | Новый файл |
| `Ctrl+O` | Открыть MD |
| `Ctrl+S` | Сохранить |
| `Ctrl+E` | Только редактор |
| `Ctrl+P` | Только просмотр |
| `Ctrl+B` | Режим «Вместе» |
| `Ctrl+Shift+B` | Жирный текст |
| `Ctrl+Shift+I` | Курсив |
| `Ctrl+Shift+U` | Подчёркнутый |
| `Ctrl+Shift+S` | Зачёркнутый |
| `Ctrl+Shift+L` | Маркированный список |
| `Ctrl+Shift+N` | Нумерованный список |
| `Ctrl+Shift+T` | Таблица |
| `Alt+левая_стрелка` | Назад в веб-странице (при переходе по ссылке) и выйти из ссылки |
| `Alt+правая_стрелка` | Вперёд в веб-странице (при переходе по ссылке)                 |

### Технологии
- `NET 8` / `WPF` — платформа разработки
- `WebView2` — рендеринг предпросмотра
- `Markdig` — парсинг Markdown
- `MailKit` / `MimeKit` — отправка email
- `highlight.js` — подсветка синтаксиса кода
